You are answering an agent's question about CatWing brand tokens. Give them the canonical value and the source-of-truth path; do not invent values, do not fall back to memory.

This skill is a flat reference card. The full brand guide is `brand.html` (compiled from `build.py`, deployed at https://catwing-repo.github.io/catwing-brand/). When the question is broader than a single token, point the asker there.

## Machine-readable tokens (preferred for programmatic use)

Both files are generated by `tools/build.py` and deployed to GitHub Pages:

- **`tokens.json`** — every color (with hex, role, group), tints, gradients, type scale, fonts, logos. Stable URL: `https://catwing-repo.github.io/catwing-brand/tokens.json`
- **`tokens.css`** — flat `:root { --cw-color-*, --cw-tint-*, --cw-gradient-*, --cw-font-* }`. Stable URL: `https://catwing-repo.github.io/catwing-brand/tokens.css`

If the asker is wiring up a Tailwind config, a CSS variable file, or a TS theme module: point them at `tokens.json` / `tokens.css`. Tables below are for human-readable lookups only.

## Typography

Canonical stacks (from `build.py::FONTS`):

| Role | Stack | Weights | Use |
|------|-------|---------|-----|
| Body | `'Poppins', -apple-system, BlinkMacSystemFont, sans-serif` | 400, 500, 600, 700 | Default UI, body text, headings |
| Display | `'Outfit', 'Poppins', sans-serif` | 700 | Logo wordmark, display headings (hero, login title) |
| Mono | `SFMono-Regular, Consolas, 'Liberation Mono', Menlo, monospace` | 400, 500 | Code blocks, hex values, data labels |

### Type scale (from `build.py::TYPE_SCALE`)

| Name | Size | Weight | Use |
|------|------|--------|-----|
| Display | 2.4rem | 700 | Login title, hero headings |
| H1 | 1.5rem | 700 | Page headings, app header |
| H2 | 1.1rem | 600 | Section headings, detail panel titles |
| Body | 1rem (14px) | 400 | Default text, paragraphs |
| Small | 0.85rem | 500 | Table cells, form labels, sidebar text |
| Caption | 0.75rem | 500 | Status card labels, footer text |
| Badge | 0.78rem | 600 | Status badges, product labels |

For Tailwind / CSS-var configs, prefer `tokens.css` `--cw-font-{body,display,mono}` over re-typing the stacks.
